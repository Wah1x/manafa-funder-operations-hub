About this tool

What this is

  • A static reference — not connected to production. No live liquidity, limit utilization, or file status is shown.
  • Timings and rules come from the sources cited on each funder profile.
  • All times are AST (Asia/Riyadh, UTC+3).

Reading a status

  • Open — a live sending window right now; funds same-day.
  • Queueing — accepted, held for the next scheduled file.
  • Closed — no dispatch today. Rejects — actively refused, not held.

Editing

  • Everything shown lives in the DATA block at the top of this file's script — edit cut-offs, holidays, contacts, and limits there; no logic changes needed.
  • Design: shadcn/ui "New York", Geist, monochrome tokens — colour is reserved for availability status.
